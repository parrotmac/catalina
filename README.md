# Usage

Either node/npm or Docker will work

### Docker:
```
docker build -t catalina .
docker run -it --rm -p 80 catalina
```
🎉 And 🎉 you're 🎉 done 🎉



### NPM/Node
```
npm run build
npm run start
```
